# 🌐 Agile Test

Bài test frontend intern 

## 🚀 Công nghệ sử dụng

- React.js
- TypeScript
- SCSS
- React Router
- Axios
- React Query
- Framer motion

## ⚙️ Hướng dẫn cài đặt và chạy dự án

```bash
git clone https://github.com/helloVietTran/agile-test-fe
cd agile-test-fe
npm run dev

```
